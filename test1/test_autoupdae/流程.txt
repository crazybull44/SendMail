如何超时不检查

数据库(版本号，上次检查，pkg)


Func Main:
    是否要检查
    {
        下载新版本号
        获取失败/超时
        {
            运行现存版本
        }
        判断是否有新版本
        {
            下载新版本(bytes_io)
            {
                保存“上次检查”，新pkg和版本号
                运行新pkg
            }
            无/获取失败/超时
            {
                运行现存版本
            }
        }
        无
        {
            保存“上次检查”,版本号    
            运行现存版本
        }
        
    }
    否
    {
        运行现存版本
    }


Func 是否要检查：
    取上次检查时间
    {    
        now > 上次 + 间隔 ？
        {
            返回检查
        }
        否则
        {
            返回不检查
        }    
    }
    取不到
    {
        返回检查
    }
